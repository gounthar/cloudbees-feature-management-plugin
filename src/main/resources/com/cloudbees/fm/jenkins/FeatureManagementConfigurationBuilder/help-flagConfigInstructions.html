Feature Flag "Set" (create and/or configure) and "Delete" instructions.

Format:
{
  set: [
    // Flag schema objects - flags to be created and/or configured
  ],
  delete: [
    // Flag name strings - flags to be deleted
  ]
}

Example - create a flag (only) i.e. not enabled etc.:

{
  set: [
    {
      name: 'acmeFlagControl'
    }
  ]
}

Example - create and configure a flag:

{
  set: [
    {
      name: 'acmeFlagControl',
      description: 'Applies foo to our Nifty App',
      enabled: true,
      labels: ['do-not-merge'],
      stickinessProperty: 'fixed'
    }
  ]
}

Example - delete flag(s):

{
  delete: ['acmeFlagControl']
}
